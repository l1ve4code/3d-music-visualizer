## Installing

**Firstly** clone the project.

```git
git clone https://github.com/l1ve4code/3d-music-visualizer.git
```

**Secondly** install dependencies
```cmd
npm install
```

**Thirdly** run project

### For watch

```cmd
npm run watch
```

### For build

```cmd
npm run build
```
